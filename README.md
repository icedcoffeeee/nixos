## steal my config
1. flash the minimal version of nixos (no desktop).
1. login and run:
    ```sh
    curl https://raw.githubusercontent.com/icedcoffeeee/nixos/main/install.sh > install.sh
    chmod +x ./install.sh
    ./install.sh
    ```
1. when vim opens, change your user and hostname.
1. wait for build (20GB build)
1. reboot!
