## steal my config
1. flash the minimal version of nixos (no desktop).
1. login and run:
    ```sh
    curl https://raw.githubusercontent.com/icedcoffeeee/nixos/refs/head/main/install.sh | bash
    ```
1. when vim opens, change your user and hostname.
